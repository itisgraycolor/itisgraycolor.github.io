AGFN: Time Windows 확장

Goal

AGFN(Adversarial GFlowNet) 프레임워크를 VRP의 Time Windows(TW) 제약으로 확장.

Key idea

  • state s에 대해 feasible mask M(s)로 TW 위반을 사전 차단
  • trajectory balance로 forward/backward flow 일치, reward는 route 비용의 음수 지수 형태 R = exp(−α·cost)
  • discriminator로 heuristic target 분포 보정

Pseudo

// feasible mask (TW)
            M(s, i) = [arrival_time_i ≤ l_i] ∧ [service_window_ok]

            // sampling
            a ~ πθ(a|s) ∝ Fθ(s→s')
            s ← s'

            // reward
            R = exp(−α · total_cost(s_T))
            L_TB = (log Fθ(forward) − log Fθ(backward) − log R)^2

Notes

  • mask가 전부 0이면 depot로 강제 복귀
  • α 스케줄은 warmup 후 cosine decay
  • 실험 세팅과 seed를 함께 공개해 재현성 확보
